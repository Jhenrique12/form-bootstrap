# Formulário com uso de Bootstrap

## Um simples formulário utilizando componentes do Bootstrap
Elementos do bootstrap que foram utilizados:

Inputs:
- Name;
- Radio;
- Switch;
- Checkbox;


Botões:
- Botão primary (com modificação nas cores do texto, da borda e de fundo);
